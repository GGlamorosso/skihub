Monorepo CrewSnow (backend Supabase ( LOUIS ) + app Flutter( RALPH ) )



# CrewSnow (Monorepo)

- **backend/** : Supabase (SQL migrations, seed, Edge Functions)
- **apps/mobile/** : App Flutter (Dart)
- **env/** : exemples de variables d'env (dev/prod)
- **scripts/** : scripts d'aide (dev, reset, checks)

## Démarrer (débutant)
1) Cloner le repo
2) Mettre vos clés locales (jamais dans Git) à partir de `env/dev/*.env.example`
3) Travailler dans une branche `feature/...`
4) Ouvrir une Pull Request pour merger dans `main`
