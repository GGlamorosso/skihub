# Backend (Supabase)

- Migrations SQL : `backend/supabase/migrations/`
- Seed : `backend/supabase/seed/`
- Edge Functions (Deno/TypeScript) : `backend/supabase/functions/`

## Démarrer (simple)
- Créer un projet Supabase (dev)
- Appliquer les migrations via l'éditeur SQL Supabase
- Importer le seed (stations) si dispo
